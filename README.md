Feedly Mac App
==============
Forget about Google Reader and run [Feedly][feedly] as a Mac app using Google Chrome.

### HOW TO INSTALL

```CLI
$ git clone git@github.com:heldr/feedly-mac-app.git
$ cd feedly-mac-app
$ make install
```
